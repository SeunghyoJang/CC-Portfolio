Project #8 Experimental Camera

code link

Click mouse to save it!

Description

In this experimental camera project, it shows the reflection on the camera with an abstract filter that flashes like a pop art. We are being captured by cameras everywhere, but we don't know how it is actually photographed. I can compare my actual appearance and what is filmed with a viewfinder. You can also check the actual time below when it is being filmed. If you put a pointer on abstract filter, you can see the actual appearance corresponding to it through the viewfinder. It is also saved automatically by clicking on the mouse.



Reflection

It was fun to make a filter. The concept is surveillance, so I want to shoot it in a different place with a different filter. Now there are filters and viewfinders in one canvas, but I thought it would be really like CCTV in the world if different people could see and store them on two other canvases.
